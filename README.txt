* Name Of project :-  Calculate Folder Stats
* Project over view :- This project take file or directory path from user in string format and give all stats in terms of complete canonical path of all files with name, size and extension type other than these funcationlity give total count specific file types in the form of key value pair and lastly gives total directory size.

* there is four methods :- calculateStats(), checkExtension(), displayDirectoryContents(), getFileFolderSize()

* User input : give complete path of folder or file in string format.

getFileFolderSize() methods get input from user in string format and return complete stats of file and folder.
